# Contoh: Mencari Hadits

## Skenario
User ingin mencari hadits tentang keutamaan ilmu beserta takhrij-nya.

---

### Percakapan

**User:** Carikan hadits-hadits tentang keutamaan menuntut ilmu

**Asisten:**

Baik, saya carikan hadits-hadits tentang keutamaan menuntut ilmu dari kitab-kitab hadits klasik.

> `turath_search { query: "فضل طلب العلم", precision: 1 }`
> `turath_search { query: "من سلك طريقا يلتمس فيه علما" }`

Saya menemukan beberapa hadits utama. Ini hasilnya:

---

## Poin Penting dari Contoh Ini

1. **Cari dengan beberapa query** — gunakan matan hadits yang dikenal + kata kunci topik.
2. **Tampilkan teks Arab + terjemahan** — selalu bilingual.
3. **Sertakan takhrij** — sebutkan kitab sumber asli hadits (Bukhari, Muslim, dll).
4. **Jujur soal kualitas hadits** — jika hadits dhaif, sebutkan. Jangan diam saja.
5. **Tawarkan syarah** — user mungkin ingin penjelasan lebih detail dari ulama.
